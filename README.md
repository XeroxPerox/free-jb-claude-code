Hello! To use this Claude Code Free Jailbroken tool, please follow these steps:
- Complete the installation of free-claude-code in GitHub(https://github.com/Alishahryar1/free-claude-code)
- Then install dependencies if you don't have them:
‘pip install uv‘
‘pip install uvicorn‘
- Sign in Groq and create an API Key
- Edit server.py with notepad or notepad ++
- Look for GROQ_API_KEY="YOUR_GROQ_API_KEY_HERE" line and change YOUR_GROQ_API_KEY_HERE to your Groq API key.

WARNING: THE API KEY THAT YOU ENTER ARE NOT GOING TO BE SENT TO ME.

- Then run eni_launcher.

Two Command Prompt(CMD) windows must show up.

1. The first one, is the proxy server activator.
It prevents Anthropic's servers to detect you are using Claude Code for free.

2. The second one, is the Claude Code Free Jailbroken window.

In a few seconds, Claude Code will load, and an automatic message will be sent:
"hello, gentleman!"
If Claude takes long to respond(or the progress text switches colors from orange to red), press ESC on your keyboard(interrupts Claude) and type "hello, gentleman!" or if it automatically shows up, send it again and he should respond.

If you get an API error, you should edit server.py with notepad or notepad ++ and add in your GROQ API Key. If you already added it, make sure it is correct, and not expired.
